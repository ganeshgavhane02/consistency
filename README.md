# Practice Workspace

This workspace contains small HTML and CSS practice files.

## About

I am Ganesh D Gavhane, and these files show my journey in learning web development. I am currently practicing:

- HTML structure and semantic markup
- CSS styling and layout
- Responsive design basics
- Simple page layouts and naming consistency

## Features

- Portfolio page with personal information and image sections
- Traffic light display demo using HTML and CSS
- Daily journal template for learning notes and code samples

## Files

- `portfolio.html` - main portfolio page
- `stle.css` - stylesheet for the portfolio
- `display.html` - traffic light display example
- `style.css` - stylesheet for the display example
- `daily-journal/` - daily journal template and workflow
- `images/` - SVG image assets used on the portfolio page

## Preview
